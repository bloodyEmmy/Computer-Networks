Фактический стандарт на основе [[Стек протоколов TCP IP]]. Используется на практике в отличие от [[Модель OSI (Основы)]].

Имеет всего ***4 уровня***:

| ***Номер уровня*** | ***Название***      |
| ------------------ | ------------------- |
| 4                  | Прикладной          |
| 3                  | Транспортный        |
| 2                  | Интернет            |
| 1                  | Сетевых интерфейсов |
